Introdução à Objeto em JavaScript


O que são?

Objeto é uma das estruturas de dados mais importantes em JavaScript
Em JavaScript, objetos são coleções de pares chave-valor.
Os objetos permitem armazenar informações relacionadas em uma única entidade. 

                    
                        // Exemplo de objeto em JavaScript
                        let carro = {
                          marca: "Toyota",
                          modelo: "Corolla",
                          ano: 2022,
                          cor: "prata",
                          ligar: function() {
                            console.log("O carro está ligado!");
                          },
                          desligar: function() {
                            console.log("O carro está desligado!");
                          }
                        };

Os objetos são usados para modelar entidades do mundo real e abstratas em código.


Propriedades de objetos

Os objetos em JavaScript são compostos por pares de chave-valor. Cada propriedade de um objeto consiste em uma chave (também chamada de nome da propriedade) e um valor.
Acessamos as propriedades de um objeto usando a notação de ponto (.) ou a notação de colchetes ([]).

Exemplo de acesso às propriedades de um objeto:

                    
                        let pessoa = {
                            nome: "João",
                            idade: 25,
                            profissao: "Engenheiro"
                          };
                          
                          console.log(pessoa.nome); // Saída: João
                          console.log(pessoa['idade']); // Saída: 25

Podemos adicionar, modificar e excluir propriedades de um objeto em tempo de execução.


Métodos de objetos

Além de propriedades, os objetos em JavaScript também podem conter métodos.
Um método é uma função que está associada a um objeto e pode ser chamada para executar uma ação específica. 

                    
                        let pessoa = {
                            nome: "Maria",
                            idade: 30,
                            saudacao: function() {
                              console.log("Olá! Meu nome é " + this.nome + " e tenho " + this.idade + " anos.");
                            }
                          };
                          
                          pessoa.saudacao(); // Saída: Olá! Meu nome é Maria e tenho 30 anos.

Dentro de um método, a palavra-chave this se refere ao próprio objeto que o contém.
Os métodos podem ser chamados da mesma forma que as propriedades, usando a notação de ponto (.) ou a notação de colchetes ([]).



Criação de Objetos

Em JavaScript, existem várias maneiras de criar objetos:
Sintaxe literal: Podemos criar objetos usando a sintaxe de chaves ({}) e definindo as propriedades e valores diretamente.
Exemplo de criação de objeto usando sintaxe literal: 

                    
                        let pessoa = {
                            nome: "Ana",
                            idade: 28,
                            profissao: "Advogada"
                          };

Função construtora Podemos criar objetos usando uma função construtora e o operador new.
Exemplo de criação de objeto usando função construtora: 

                    
                        function Pessoa(nome, idade, profissao) {
                            this.nome = nome;
                            this.idade = idade;
                            this.profissao = profissao;
                          }
                          
                          let pessoa = new Pessoa("Pedro", 35, "Engenheiro");

Classe: A partir do ECMAScript 2015 (ES6), também podemos criar objetos usando a sintaxe de classe em JavaScript.
Exemplo de criação de objeto usando classe: 

                    
                        class Pessoa {
                            constructor(nome, idade, profissao) {
                              this.nome = nome;
                              this.idade = idade;
                              this.profissao = profissao;
                            }
                          }
                          
                          let pessoa = new Pessoa("Mariana", 32, "Médica");



Protótipos

Em JavaScript, protótipo é um mecanismo que permite a herança entre objetos.
É um mecanismo que permite a herança entre objetos.
Cada objeto em JavaScript tem um protótipo, exceto o objeto raiz (Object.prototype).
Um objeto herda propriedades e métodos de seu protótipo.
A herança de protótipos permite compartilhar propriedades e métodos com eficiência entre objetos.
Os métodos são definidos como propriedades que possuem valores de função. 

                    
                    // Objeto protótipo
                    let animal = {
                    fazerBarulho: function() {
                        console.log("O animal faz barulho.");
                    }
                    };

                    // Objeto que herda do protótipo
                    let cachorro = Object.create(animal);
                    cachorro.fazerBarulho(); // Saída: O animal faz barulho.



Iteração de Objetos

Em JavaScript, podemos percorrer as propriedades de um objeto usando loops e métodos específicos.
Loop for...in: Podemos usar o loop for...in para iterar sobre as propriedades de um objeto. 

                    
                        let pessoa = {
                            nome: "Ana",
                            idade: 28,
                            profissao: "Advogada"
                          };
                          
                          for (let propriedade in pessoa) {
                            console.log(propriedade + ": " + pessoa[propriedade]);
                          }



JSON (JAVASCRIPT OBJECT NOTATION)

JSON é um formato de dados popular utilizado para troca de informações estruturadas
JSON é baseado na sintaxe de objetos em JavaScript.
Em JavaScript, podemos converter objetos em JSON usando o método JSON.stringify(). 

                    
                        let pessoa = {
                            nome: "Maria",
                            idade: 30,
                            profissao: "Advogada"
                          };
                          
                          let jsonPessoa = JSON.stringify(pessoa);
                          console.log(jsonPessoa); // Saída: '{"nome":"Maria","idade":30,"profissao":"Advogada"}'

Em JavaScript, podemos converter JSON em objetos usando o método JSON.parse() 

                    
                let jsonPessoa = '{"nome":"Maria","idade":30,"profissao":"Advogada"}';

                let pessoa = JSON.parse(jsonPessoa);
                console.log(pessoa.nome); // Saída: Maria

JSON é amplamente utilizado para comunicação de dados entre servidores e clientes.



Objeto como atributo de objeto

Em JavaScript, é possível ter objetos como atributos de outros objetos, o que nos permite criar estruturas mais complexas e aninhadas.
Veja um exemplo de objeto que possui um objeto como atributo: 

                    
                        let pessoa = {
                            nome: "João",
                            endereco: {
                              rua: "Rua A",
                              numero: 123,
                              cidade: "São Paulo"
                            },
                            idade: 30
                          };

Nesse exemplo, o objeto pessoa possui três atributos: nome, endereco e idade. O atributo endereco é por si só outro objeto, contendo os atributos rua, numero e cidade.

Podemos acessar os atributos de um objeto aninhado utilizando a notação de ponto ou a notação de colchetes: 

                    
                    console.log(pessoa.nome); // Saída: João
                    console.log(pessoa.endereco.rua); // Saída: Rua A
                    console.log(pessoa["endereco"]["cidade"]); // Saída: São Paulo
